ENGLISH:

To open the app simply click the .exe file. 

*If the program doesn´t work correctly on the first try, close and open it again*

Controls:

 - 'W'/'UP' -> Move fowards
 - 'S'/'DOWN' -> Move backwards
 - 'A'/'LEFT' -> Move left
 - 'S'/'RIGTH' -> Move right
 - Use the mouse to look around

 - 'SPACE' -> Turn on/off planet movement
 - 'E' -> Turn on/off real scale
 - 'R' -> Reset everything
 - 'LEFT SHIFT' -> Move faster
 - 1-9 -> Hold to see the info about the planet
 
 -There´s an hidden entity somewhere in the galaxy and a forgotten secret.....
 
 --------------------------------------------------------------------------------------------------------------------
 
 To run this program you´ll need to install GLFW 32bit, GLEW 32bit and GLM 32bit, after that, configure such libraries in the project.
 
 It is also needed to configure the pre-processor by implementig '_CRT_SECURE_NO_WARNINGS'.
 
 Add Winmm.lib in Linker->input-> Additional libraries
 
 And that´s it. Enjoy!
